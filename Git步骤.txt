1 安装Git for Windows
2 在GitHub注册自己的账号
3 Config global account
  git config --global user.name "sure.deng"
  git config --global user.email "sure.j.deng@newegg.com"
4 Generate SSH key
  ssh-keygen -t rsa -C "sure.j.deng@newegg.com"
5 Copy SSH Key
  clip < ~/.ssh/id_rsa.pub
6 Test connection
  ssh -T git@github.com
7 git clone "http://"
8 git push origin master
9 改变Remote Url：
  git remote set-url origin "git@github.com:user/responsitory.git"
  git remote -v
10 定义Git 比较工具
  git config --global diff.tool bc3
  git config --global difftool.bc3.path "C:\Program Files (x86)\Beyond Compare 3\BCompare.exe"
  git config --global merge.tool bc3
  git config --global mergetool.bc3.path "C:\Program Files (x86)\Beyond Compare 3\BCompare.exe"
11 如何比较两次Commit之间的差异
  git diff Commit1 Commit2 / git difftool commit1 commit2
